🧠 Why This Stack For Xianxia Donghua?

Donghua's signature look = 2D energy + 3D environment + real-time rendering. Niagara handles particles & ribbons, Materials handle the glow & dissolve magic, Krita handles the ink-brushstroke flipbooks, and UE5 Lumen handles the lighting that makes everything pop. This is exactly the pipeline used in modern stylized games.

⚠️ The Golden Rule — Escape Tutorial Hell

For every 1 tutorial you follow, you must build 1 original xianxia variant. Tutorial beam → your tribulation lightning. Tutorial aura → your golden core aura. Tutorial portal → your sect formation gate. This is how you actually get good, not just informed.

Daily

Your Full-Day Learning Formula

🌅 Morning · 3–4 Hours
📚
Learn Mode
  • Collect reference screenshots from donghua shows
  • Break down the effect: what layers does it have?
  • Watch the tutorial, take notes on key nodes
  • Follow along step-by-step in UE5
☀️ Afternoon · 3–4 Hours
🔨
Build Mode
  • Rebuild the effect WITHOUT watching — from memory
  • Change color, timing, scale: make it YOUR version
  • Make it donghua/xianxia flavored
  • Break stuff on purpose — learn each parameter
🌙 Evening · 2–3 Hours
✨
Polish & Post Mode
  • Polish daily output: color, timing, size
  • Record a short clip or GIF of the effect
  • Post it: Twitter/X, Reddit r/unrealengine, Discord
  • Save to "VFX Library" folder → future FAB pack
📸 Every Single Day
🎯
The Non-Negotiable Rule
  • Name your file: "Day_XX_EffectName"
  • One finished, polished output — no exceptions
  • Post publicly — accountability builds momentum
  • Mark the day complete in the tracker ↑
